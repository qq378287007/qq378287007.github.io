命令

  • M x y
    • moveto
  • L x y
    • lineto
  • H x
    • horizontal lineto
  • V y
    • vertical lineto
  • C x1 y1 x2 y2 x y
    • curveto，起点：当前位置，终点：x y，起始控制点：x1 y1， 终点控制点：x2 y2，三次贝塞尔曲线
  • S x2 y2 x y
    • smooth curveto，跟在C或S指令后面，自动生成一个对称点x1 y1
  • Q x1 y1 x y
    • quadratic Bézier curve，起点：当前位置，终点：x y，控制点：x1 y1，二次贝塞尔曲线
  • T x y
    • smooth quadratic Bézier curveto，跟在Q或T指令后面，自动生成一个对称点x1 y1
  • A rx ry x-deg large-arc sweep-flag x y
    • 圆弧
    rx ry表示x轴半径和y轴半径（看比值）
    x-deg表示x轴旋转角度（顺时针为正）
    前三个参数确定时，满足当前点到指定点(x y)的圆弧有四条
    large-arc表示优劣弧(0为劣弧，1为优弧)
    sweep-flag表示弧线方向(0为逆时针，1为顺时针)
    x y为最终坐标
  • Z
    • closepath，终点和起点绘制直线
以上所有命令均允许小写字母。大写表示绝对定位，小写表示相对定位。
