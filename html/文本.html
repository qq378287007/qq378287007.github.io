块、行内元素

（div）块元素独占整一行，内部可容纳块元素和行内元素。
（span）行内元素可与其他行内元素位于同一行，内部只可容纳行内元素。

统一资源定位器(Uniform Resource Locators)

scheme://host.domain:port/path/filename
  • scheme
    • 服务类型
  • host
    • 域主机
  • domain
    • 域名
  • :port
    • 主机端口号, http默认80
  • path
    • 服务器上的路径
  • filename
    • 文档/资源的名称

URL字符编码

URL只能使用ASCII 字符集.
URL 编码使用"%"其后跟随两位的十六进制数来替换非ASCII字符。
