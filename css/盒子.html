定义

所有元素都可以看成一个盒子。
盒子组成
属性 说明
content 文本或图片，width、height、overflow（溢出处理方式）
padding 内容与边框距离，padding-top|bottom|left|right
margin 当前元素与其他元素间的距离，margin-top|bottom|left|right，父子或兄弟元素
border 元素边框，bodrer-width|style|color
div{ display:inline-block;/*块元素转换为inline-block元素，宽度由内容区撑起来*/ padding:20px; margin:40px; border:2px solid red; background_color:#FFDEAD; }

宽高

宽高针对内容区，只有块元素（加上inline-block元素）才可以设置width和height。
dispaly属性可以实现块元素和行内元素的转换。 div{ width:100px;/*块元素不设置width时，延伸一整行*/ height:50px; border:1px solid red; } span{ width:100px;/*行内元素width和height失效，宽高由内容区撑起来*/ height:50px; border:1px solid red; }

边框

div{ border:1px solid red; }

内边距

div{ display:inline-block;/*块元素转换为inline-block元素，宽度由内容区撑起来*/ padding-top:20px; padding-right:40px; padding-bottom:60px; padding-left:80px; border:2px solid red; background_color:#FFDEAD; } padding:20px; padding:20px 40px;/*top right bottom left顺序*/ padding:20px 40px 60px; padding:20px 40px 60px 80px;
